## Example of using MDTextFieldRound:

```python
            MDTextFieldRound:
                hint_text: 'Password'
                icon: 'lock-outline'
```

## Parameters:

* width - field width
* icon_color - color of right icon
* active_color - color of field when on focus
* normal_color - color of field when not focus

## Same options as in TextInput:
* foreground_color
* hint_text_color
* cursor_color
* selection_color
